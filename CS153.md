# CS153 submission quick reference

Full documentation: **[README.md](./README.md)**

## Grader checklist (in-app)

| Requirement | Tab / action |
|-------------|----------------|
| AI diligence | Feed → **View diligence** → scroll memo sections + score |
| Forecasting | **Forecast** on card or detail |
| Investment decisions | **Invest** / **Pass** + conviction |
| Leaderboard / reputation | **Leaderboard** tab |
| Founder submission | **Submit** tab → form → generated diligence |

## Rubric (1–5)

1. **Problem & Insight** — Feed mission line + README “What the app does”
2. **Execution & Technical Work** — Expo/TS app, AI pipeline, optional Supabase, demo mode
3. **Evaluation & Evidence** — Leaderboard + profile stats/activity after your session actions
4. **Communication** — Capability pills on Feed, README demo script
5. **Process & Integrity** — Profile “CS153 · Process & disclosure”; README tooling section

## Run (no keys)

```bash
npm install && cp .env.demo .env && npm run start:demo
```
